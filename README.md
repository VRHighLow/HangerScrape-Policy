# HangerScrape-Policy
Privacy Policy – Star Citizen Hangar Sync
Effective: 2025-12-05
Contact: thedisplaystudio@gmail.com

What this extension does

Reads your own hangar pages on https://robertsspaceindustries.com to capture pledge data (pledge names, items/contains, exchange/melt values, insurance, hangar location, page number, status, and your RSI handle as displayed).
Data we collect

Website content from the RSI hangar page only (as listed above).
No passwords, payment data, or other personal identifiers are collected.
How data is used

To show your hangar data in the popup and let you download it as JSON.
To upload to your configured server endpoint if you trigger upload.
Storage & sharing

Data is stored locally in the extension’s storage.
Remote upload occurs only if you trigger it, to the server URL you configure.
Data is not sold or shared with third parties.
Permissions (why they’re needed)

activeTab, scripting: inject a content script into your RSI hangar tab to read pledge data.
storage: save captured hangar data locally so you can view/download without re-scraping.
notifications: only to notify you if a website-triggered sync can’t auto-open the popup.
Host permission https://robertsspaceindustries.com/*: required to read the hangar page content.
Retention & control

Local data stays until you clear it or uninstall the extension.
Remote data retention is controlled by you/your configured server.
Security

No credentials are collected. Data is read from RSI and sent only to your chosen upload endpoint when you trigger it.
Children

Not intended for children under 13.
Changes

Updates will be posted at this URL with a new effective date.
Contact

For questions or requests: thedisplaystudio@gmail.com.
